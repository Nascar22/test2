ARK Blockchain Framework

ARK Core v3 is the world's most open and adaptable framework for building Blockchain applications. Built on Typescript with connections to other popular languages and technologies.
Open Source ARK is fully open source allowing anyone to contribute. Earn bounties for your contributions and join a thriving community of developers just like you.
Modular Architecture The ARK Blockchain Framework architecture is fully modular, allowing you to customize your chain to fit your application's needs.
ARK Logic - Custom Smart Transactions ARK Logic allows you to create your own custom Smart Transactions that allow your application to implement native Smart Contract functionality in a simple way.
ARK HTLC SmartBridge ARK HTLC SmartBridge allows for the exchange of value and data between blockchains using ARK's custom Hashed Time Lock Contracts transaction logic.
