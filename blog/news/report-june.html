June was fantastic! We managed to update our desktop & mobile apps, have AMA session with Justin Sun, the Neo’s team and be guests at the Cosmos’ and ChangeNOW communities, add ICON (ICX), finally, AWC staking was bringing the first outstanding results! Welcome to read it up!


Desktop Release (2.17.0)

The new desktop version 2.17.0 has these features:

  • New price chart section. Stay tuned with the market trends.
  • ICON (ICX) added. Manage and exchange ICX in a secure way.
  • AWC staking with a 23% yearly yield.
  • Band Protocol (BAND) added with staking support.
  • Instant balance updates for ATOM and TRX are speeded up.

You can download Atomic here.


Mobile Release (0.61.0)

The new mobile version 0.61.0 has these features:

  • Added ICON (ICX) for holding and exchange. ICX staking is soon.
  • AWC is available for staking with a 23% max yearly profit.
  • Buy crypto history now syncs properly between devices.
  • Band Protocol (BAND) added with staking support.
  • JUST Airdrop added to the staking section.
  • Bug fixes for KMD staking, BTC, ETH & ERC20 tokens fees calculation.

You can download Atomic here.


AWC Staking

Atomic Wallet Token (AWC) staking has been out for 5 weeks by now! The latest data is 2,540,721 AWC ($1,6M+) staked by 2662 holders while we have already distributed 32,663 AWC ($21,000+) rewards. Start staking from a minimum of 10 AWC and get rewards up to 23% yearly profit. So far so good! Read more about the token.


Membership Program

One more reason to hold AWC is new rules for our loyalty program! Now the Basic level starts at 300 AWC. You can earn up to 1% cashback for any exchange or crypto purchase in Atomic! Become a member!
